## Cattle & Buffalo Breed Recognition System

An AI-powered web application that identifies cattle and buffalo breeds from images using a deep learning model (EfficientNetV2-M). The system provides top-5 predictions with confidence scores and maintains prediction history.

---

## 🚀 Features

- 🔍 Image-based breed recognition
- 🧠 Deep learning model (EfficientNetV2-M)
- 📊 Top-5 predictions with confidence scores
- 🕒 Prediction history tracking
- 🌐 User-friendly web interface (Flask/React)
- ⚡ Fast and efficient inference

---

## 🏗️ Tech Stack

- **Frontend:** React.js / HTML / CSS
- **Backend:** Flask (Python)
- **Model:** EfficientNetV2-M
- **Libraries:** TensorFlow / PyTorch, NumPy, OpenCV
- **Deployment:** Local / Cloud (optional)
